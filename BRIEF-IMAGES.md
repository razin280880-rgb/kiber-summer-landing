# ТЗ дизайнеру: 11 фото для лендинга kanikuly.it-kiber.ru (КИБЕРканикулы)

## Общие правила (читать ОБЯЗАТЕЛЬНО)

1. **Формат:** JPEG, sRGB, прогрессивный. Сжатие — до 250 КБ каждый файл (баннеры до 400 КБ). Без водяных знаков.
2. **Имена файлов — точно как ниже.** Иначе сайт не подхватит.
3. **Способ создания:** AI-генерация (Midjourney v6/v7, DALL-E 3, Sora, Flux, Recraft) — промпт прилагается. Альтернатива — реальная съёмка в наших филиалах с письменным согласием родителей (152-ФЗ).
4. **Главное правило по лицам детей:** лица **не должны быть узнаваемы**. Приёмы:
   - вид со спины / в три четверти со склонённой головой,
   - крупный план — руки на клавиатуре, голова обрезана,
   - лёгкий motion blur,
   - в AI-промпте — `face slightly blurred, soft focus on face`.
5. **Бренд-палитра KIBERканикул (сайт тёмный!):**
   - Базовый фон сайта — `#07080F` (тёмно-синий-чёрный)
   - Акцент-синий — `#2563EB`
   - Акцент-фиолетовый — `#8B5CF6`
   - Cyan — `#06B6D4`
   - Оранжевый (горящий лагерь) — `#F97316`
   - Жёлтый — `#FBBF24`
6. **Стиль:** яркий, насыщенный, контрастный. Сайт тёмный — поэтому фото должны хорошо смотреться на тёмном фоне. **Не пастель, не приглушённые цвета.** Фото с насыщенной экспозицией, чёткими акцентами цвета (синие толстовки, оранжевые рюкзаки, цвета на экранах). Атмосфера «лето + AI + энергия».
