# KIBERone — Летний AI-лагерь

Одностраничный лендинг летнего лагеря KIBERone «КИБЕРканикулы».

- **Стек:** чистый HTML5 + CSS + vanilla JS, без сборки.
- **Аналитика:** Яндекс.Метрика, VK Pixel.
- **Лиды:** AlphaCRM (форма захвата).
- **Хостинг:** Cloudflare Pages, домен `kanikuly.it-kiber.ru`.

## Деплой

Автоматический: push в `main` → Cloudflare Pages пересобирает сайт.

## Локальный просмотр

Открой `index.html` в браузере.
